# Security Policy

## Supported Versions

Only the latest published version is supported with security updates.

## Reporting a Vulnerability

Please report vulnerabilities privately.

Preferred channels:

1. GitHub Security Advisory (private reporting on this repository)
2. Email: `dobroslav.radosavljevic@gmail.com`

Please include:

- A clear description of the issue
- Reproduction steps or proof of concept
- Affected package version(s)
- Any suggested mitigation

## Response Targets

- Initial acknowledgment: within 3 business days
- Triage and severity assessment: within 7 business days
- Fix timeline: depends on severity and complexity, communicated during triage

Please do not disclose the issue publicly until we confirm a fix is available.
